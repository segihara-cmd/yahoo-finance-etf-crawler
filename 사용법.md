# Yahoo Finance ETF 크롤러 사용법

## 일상적인 사용

### 1. 크롤링 실행
```
실행.bat 더블클릭
```
→ ETF 데이터를 크롤링하여 엑셀 파일로 저장합니다.

### 2. 코드 수정 후 GitHub 업로드
```
git_빠른푸시.bat 더블클릭
```
→ 수정한 코드를 자동으로 GitHub에 업로드합니다.

## 처음 사용하는 경우

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 크롤링 실행
```
실행.bat 더블클릭
```

## GitHub 업로드

코드를 수정한 후:
1. `git_빠른푸시.bat` 실행
2. Personal Access Token 입력 (처음 한 번만)
3. 완료!

## 파일 구조

- **크롤링 관련**: `crawl_etf_gainers.py`, `crawl_etf_gainers_selenium.py`
- **실행**: `실행.bat`
- **GitHub 업로드**: `git_빠른푸시.bat`
- **설정**: `requirements.txt`, `.gitignore`

## 주의사항

- 엑셀 파일(`*.xlsx`)은 GitHub에 업로드되지 않습니다 (`.gitignore`에 의해 제외)
- Personal Access Token은 GitHub → Settings → Developer settings에서 생성할 수 있습니다
